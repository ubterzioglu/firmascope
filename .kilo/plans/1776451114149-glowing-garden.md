## Faz 14: Edge Case Testleri

### 14.1 Auth Edge Cases
- [ ] **Token expire:** Session expire sonrası auth state güncellemesi
- [ ] **Concurrent render:** Birden fazla `useAuth` consumer aynı anda
- [ ] **Network hatası:** Supabase çağrıları network error dönerse
- [ ] **Boş email/şifre:** Submit butonu disabled değil ama validasyon çalışır

### 14.2 Form Edge Cases
- [ ] **Çok uzun input:** maxLength sınırları çalışır
- [ ] **Sadece whitespace:** Trim validasyonu
- [ ] **Negatif maaş/sayı:** Number input validasyonu
- [ ] **XSS input:** Script tag'leri encode edilir
